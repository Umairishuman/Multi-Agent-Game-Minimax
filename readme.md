# Project Structure

## Folder and File Organization

```
.
├── .git/                          # Git repository metadata
├── .gitignore                     # Git ignore file
├── Assignment3_GamePlaying.pdf    # Assignment document
├── readme.md                      # This file
├── requirements.txt               # Python dependencies
├── input/
│   └── board.txt                  # Input board file
├── myenv/                         # Python virtual environment
├── output/                        # Output directory (empty)
└── src/
    ├── __init__.py
    └── main.py                    # Main application file
```

## Directory Descriptions

| Directory | Purpose                                      |
| --------- | -------------------------------------------- |
| `.git/`   | Git version control metadata                 |
| `input/`  | Input data files for the application         |
| `myenv/`  | Python virtual environment with dependencies |
| `output/` | Output directory for application results     |
| `src/`    | Source code directory                        |

## Key Files

| File                          | Description                       |
| ----------------------------- | --------------------------------- |
| `readme.md`                   | Project documentation             |
| `requirements.txt`            | Python package dependencies       |
| `Assignment3_GamePlaying.pdf` | Assignment specification document |
| `src/main.py`                 | Main application script           |
| `input/board.txt`             | Input board data                  |
